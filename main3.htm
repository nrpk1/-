2.2 Якими засобами можна створити малюнок у редакторі опис "Панель инстументов"???
Для створення малюнка і для втілення Ваших художніх талантів є різні засоби але для цього використовуєте програму Paint. У ньому були зібрані на перший погляд прості але водночас складні функції а саме я про них опишу нижче. Для цього Вам потрібно: Гумка призначена для стирання частини малюнка до кольору фону. Олівець призначений для малювання ліній. Вибирають натисканням кнопкукі Олівець. Лінія. Цей інструмент призначений для побудови прямих ліній.
